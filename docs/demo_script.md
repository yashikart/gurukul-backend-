# Gurukul Demo Script

## Introduction (1 minute)

**Opening:**
"Welcome to Gurukul - a purposeful learning platform designed to guide students through a structured, meaningful educational journey."

**Key Philosophy:**
- Not just another edtech tool
- Focused on calm, intentional learning
- Guides students through: Enter → Learn → Practice → Reflect → Improve

---

## Student Experience (5 minutes)

### 1. Dashboard & Journey Guidance (1 min)
**Show:**
- Clean, calm interface (soft colors, purposeful design)
- "Next Step" card guiding the student
- Learning journey visualization
- Progress tracking widgets

**Say:**
"Students are never lost. The dashboard always shows them what to do next in their learning journey."

### 2. Chatbot (1 min)
**Demo:**
- Ask a question about a subject
- Show AI-powered response
- Highlight conversation history

**Say:**
"Students can ask questions anytime and get immediate, helpful answers from our AI tutor."

### 3. Document Summarizer (1 min)
**Demo:**
- Upload a PDF
- Show AI-generated summary
- Generate flashcards from the document

**Say:**
"Transform any learning material into digestible summaries and study aids."

### 4. Test Generator (1 min)
**Demo:**
- Create a quiz on a topic
- Show question generation
- Take the quiz and see results

**Say:**
"Students can test their knowledge on any subject, at any difficulty level, instantly."

### 5. Flashcards (1 min)
**Demo:**
- Review flashcards
- Show spaced repetition system
- Demonstrate confidence rating

**Say:**
"Spaced repetition ensures students master concepts over time, not just memorize for tests."

## Q&A Preparation

**Common Questions:**

**Q: How is this different from other learning platforms?**
A: Gurukul focuses on the learning journey, not just features. Every element guides students toward meaningful learning, with a calm, purposeful interface.

**Q: Can it integrate with our existing systems?**
A: Yes, Gurukul uses standard authentication (Supabase) and can be integrated with existing institutional systems.

**Q: What about data privacy?**
A: All student data is secure, with role-based access control ensuring only authorized users can view sensitive information.

**Q: Is it mobile-friendly?**
A: Absolutely. Gurukul works seamlessly on phones, tablets, and desktops.

**Q: Can we customize it for our institution?**
A: Yes, the platform supports multi-tenancy and can be customized for your institution's needs.

---

## Demo Tips

1. **Keep it calm** - Match the platform's philosophy in your presentation
2. **Focus on journey** - Always bring it back to the learning path
3. **Show, don't tell** - Let the interface speak for itself
4. **Highlight purpose** - Explain WHY each feature exists
5. **Be authentic** - This is about real learning, not sales
