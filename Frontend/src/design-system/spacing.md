# Spacing System

This document outlines high-density spacing guidelines for optimal executive cognition on the Drishti Dashboards.

## Grid Gap Sizes
- **Layout Gap**: `16px` (`gap-4`) - Gap between major zones and grid segments.
- **Inner Widget Padding**: `16px` (`p-4` or `p-5`) - Padding within widgets for metric alignment.
- **Compact Metric Gap**: `8px` (`gap-2`) - Spacing between labels and metric values.
- **Section Margin**: `24px` (`mb-6`) - Spacing separating logical horizontal bands.
