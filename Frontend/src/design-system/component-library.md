# Component Library Definition

Drishti reusable cards and widgets specifications.

## Metrics & Indicator Cards
- **KPI Card**: Displays a single indicator score with status-color mappings, trend icons, and inline sub-details.
- **Executive Metric Card**: High-density compact view displaying multiple related scores in single container.
- **Telemetry Card**: Displays active continuous metrics like resource usage percentages (CPU, Memory).

## Signals & Feedback Cards
- **Alert Card**: Real-time signal alert displaying category, priority label, status toggles (OPEN/RESOLVED/CLOSED), and owner assignments.
- **Action Card**: Recommended governance workflows presenting task descriptions, status dropdowns, and target assignees.
- **Timeline Card**: Sequential feed item illustrating status transitions, logs, or user audit trails.

## Visualization & Geo Cards
- **Map Card**: Container wrapping spatial leaflet overlays and drilldown controls.
- **Trend Card**: Container wrapping Apache ECharts lines, bars, or donut visualization assets.
- **Recommendation Card**: Explains automated suggestions (e.g. remedial scheduling) based on cognitive telemetry.
