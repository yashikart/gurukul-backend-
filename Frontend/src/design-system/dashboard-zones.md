# Dashboard Layout Zones

The standardized grid canvas partition rules for Drishti Governance Dashboards.

## Core Grid Zones
1. **Header Zone**: Positioned top. Displays pathing context, switch level menu, and systemic ping logs.
2. **KPI Band**: A compact, horizontal ribbon showing top high-level metrics.
3. **Primary Content / Visual Zone**: Center section showing visual data representation, heatmaps, or mapping.
4. **Operations Log & Queue**: A side layout slot displaying alerts and active recommended actions.
5. **Trace Telemetry Terminal**: Located at the bottom for raw audit-logs and system-monitor outputs.
