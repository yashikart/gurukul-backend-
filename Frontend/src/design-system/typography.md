# Typography System

Typography defaults for high cognitive scan rates in Drishti Dashboards.

## Hierarchy & Sizing
- **Dashboard Title**: `28px` to `36px` (`text-3xl` / `text-4xl`), font-weight: `800` (`font-bold`), tracking-tight.
- **Widget Section Header**: `14px` (`text-sm`), font-weight: `700` (`font-bold`), tracking-wide, uppercase, tracking-widest.
- **KPI Value**: `30px` to `36px` (`text-3xl` / `text-4xl`), font-weight: `800` (`font-bold`), font-mono for numeric alignments.
- **Body / Telemetry Data**: `12px` (`text-xs`), font-weight: `500` (`font-medium`), custom color variants.
- **Console Log / Code Symbol**: `10px` (`text-[10px]`), font-mono.
