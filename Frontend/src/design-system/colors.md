# Color System

This document defines the semantic color system for the Drishti Dashboard Capability Layer.

## Core Palette
- **Background (Deep Space)**: `#050505` - Main canvas for all dashboard interfaces.
- **Glass Panel Fill**: `rgba(255, 255, 255, 0.03)` with `backdrop-filter: blur(12px)`.
- **Glass Panel Border**: `rgba(255, 255, 255, 0.08)`.

## Semantic Status Colors
- **Healthy / Compliant (Emerald)**: `#10b981` (CSS classes: `text-emerald-400`, `bg-emerald-500/10`)
- **Signal Anomaly / Risk (Amber)**: `#f59e0b` (CSS classes: `text-amber-400`, `bg-amber-500/10`)
- **Critical / Anomaly (Rose)**: `#f43f5e` (CSS classes: `text-rose-400`, `bg-rose-500/10`)
- **Info / Neutral (Sapphire)**: `#3b82f6` (CSS classes: `text-blue-400`, `bg-blue-500/10`)
